# AI README (Default Guidance)

- Canonical documentation lives in `docs/`. Root files should be short pointers only.
- Key pointers: `README.md` (overview), `docs/README.md` (index), `docs/SETUP_GUIDE.md`, `docs/PRD.md`, `docs/Handover.md`, `CHANGELOG.md`.
- When in doubt, prefer reading pointers and canonical docs over large code dumps.
- Use PACT (Plan, Act, Check, Tell) across all agents and tools.
- Follow project conventions: no inline CSS, run lint/type-check/build, keep diffs focused.
